## Installation Instruction

1. [Download](https://github.com/ahanafi/sistem-penilaian-guru/archive/refs/heads/master.zip) or clone this repository to your Computer.
2. After that, extract the files to Php project path, like **C:/xampp/htdocs** if you are using Windows and XAMPP. But, if you use Linux,
   you can extract files to **/var/www/html/**
3. Rename project directory to **skp-system**
4. Open terminal in the project directory and install dependency library using command : `bash composer install` or  `bash composer update`
5. Import the database file that is in the database folder.
6. Run in browser [http://localhost/skp-system](http://localhost/skp-system])
7. Any problem? Let`s get in touch [ahanafi.id@gmail.com](mailto:ahanafi.id@gmail.com)
